IMS Global CPv1.2 CCv1.3 Profile ResourceType Vocabulary

Vocabulary Details

This document provides a human readable description of the vocabulary terms defined in the IMS Global Vocabulary Definition and Exchange (VDEX) file "ccresourcetypevocabularyv1p3.xml". The details of the vocabulary are:

Vocabulary Identifier: http://www.imsglobal.org/xsd/cc/ccv1p3/ccresourcetypevocabularyv1p3
Vocabulary Type: flatTokenTerms
Registered Status: true
Vocabulary Language: en-US
Author: Colin Smythe, IMS Global (UK)
Issue Date: 2013-03-31
Version: Final 1.3
Status: Final Release
Base Specification: Common Cartridge (Content Packaging) Version 1.3
Keywords: Common Cartridge, Resource Types

Description: The vocabulary for the permitted resource types in the CCv1.3.
History: Version 1.1: The first release in which the vocabularies were defined as an external VDEX instance. A new resource type to identify Basic LTI v1.0 resources has been added in this version. Version 1.2: Just date and version numbering changes. Version 1.3: Addition of the new resource types APIPv1.0, IWBv1.0 and EPUBv3.0. Support for extension resource types.

Vocabulary Terms

The set of terms for the vocabulary are described in the table below.

Identifier Caption Status Description
assignment_xmlv1p0 Assignment Provisional This is used when a resource is an Assignment. The schema and schema version elements in the resource metadata will provide further validation information.
associatedcontent/imscc_xmlv1p3/learning-application-resource Associated Content Resource Core This is the resource type of Associated Content. Typically used assets such as images, video, voice files, etc.
idpfepub_epubv3p0 IDPF EPUB v3.0 Resource Core This is the resource type of EPUBv3.0 This allows an EPUB file to be a resource in the cartridge.
imsapip_zipv1p0 Accessible Portable Item Protocol (APIP) v1.0 Resource Core This is the resource type of APIPv1.0 This allows an APIP zip file to be a resource in the cartridge.
imsbasiclti_xmlv1p3 Basic LTI v1.0 Resource Core This is the resource type of Basic LTI. This allows LTI links to be embedded in a cartridge.
imsdt_xmlv1p3 Discussion Topic Resource Core This is the resource type of Discussion Topic. This enables the package to include discussion topics.
imsiwb_iwbv1p0 Interactive White Board (IWB) v1.0 Resource Core This is the resource type of IWBv1.0 This allows an IWBv1.0 zip file to be a resource in the cartridge.
imsqti_xmlv1p2/imscc_xmlv1p3/assessment QTI Assessment Resource Core This is the resource type of QTI Assessment. This must be used for tests and quizzes based upon QTIv1.2.1.
imsqti_xmlv1p2/imscc_xmlv1p3/question-bank QTI Question Bank Resource Core This is the resource type of QTI Question Bank. This must be used for Question Banks and these must be based upon QTIv1.2.1.
imswl_xmlv1p3 Web Links Resource Core This is the resource type of Web Links. This enables the package to include external web links.
webcontent Web Content Resource Core This is the resource type of Web Content. Typically used for HTML and PDF files.

In the above Table, the possible values and interpretation for the status field are:

  • "Final" - an IMS Global term that has been approved by the corresponding IMS APMG;
  • "Draft" - an IMS Global term that has not yet been approved by the corresponding IMS APMG;
  • "Core" - an IMS Global term that has been approved by the corresponding IMS APMG (used to differentiate from vocabs containing extension terms);
  • "Provisional" - an extension term that has not yet been approved by the corresponding IMS APMG but which is available on a provisional basis;
  • "Approved" - an extension term that has been approved by the corresponding IMS APMG.
