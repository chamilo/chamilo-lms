# CourseBlock

CourseBlock adds course-specific content blocks to footer-related plugin regions.

## Usage

1. Install and enable the plugin.
2. Assign it to one or more regions in Administration > Plugins > Regions:
   - `pre_footer`
   - `footer_left`
   - `footer_center`
   - `footer_right`
3. Open a course as a teacher or administrator.
4. Open the course settings page.
5. Fill the CourseBlock fields:
   - Pre footer
   - Footer left
   - Footer center
   - Footer right
6. Save the course settings.
7. Open the course homepage.

The plugin renders the configured content only in a course context and only when the plugin is active.
