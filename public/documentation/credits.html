Credits

In the past, we maintained a list of contributors to the Chamilo project along with a description of their contribution. This list is still available here, but as the project grew in size and complexity, maintaining this list became more and more time-consuming to the point we could not maintain it anymore.
Today's contributions are mainly obtainable through several platforms' logs:
  • Code contributions can be found in Github's list of commits for the corresponding branch.
  • Documentation contributions can be found in each guide's appendix in our online documentation.
  • A log of translator contributions has not been maintained for personal data ownership reasons when transfering the responsibilities from the Chamilo Association to BeezNest in late 2025, and the translations contribution opportunity has been discontinued and replaced by AI. We are deeply greatful to all translations contributors in the past, but Chamilo's translation files now being more than 100k words multiplied by roughly 50 languages, we think we're doing a good service to anyone by delegating this task to AI. If you spot a mistake, please report it as an issue in our tracking system.
